# Bot Army Shell - GitHub Release Workflow

## Tagging

```bash
# Create and push tag
git tag -a v0.1.0 -m "Release v0.1.0"
git push origin v0.1.0
```

## Release Notes

Include:
- Feature list
- Breaking changes
- Upgrade instructions

Binary releases for:

- macOS (amd64, arm64)
